1
DOWNLOAD LEMUR BROWSER
Install Lemur Browser from Google Play Store.
It supports Chrome extensions on Android natively.
2
DOWNLOAD THE EXTENSION
Tap the download button above.
Save the .zip file to your device storage.
3
OPEN LEMUR EXTENSIONS
In Lemur Browser go to lemur://extensions
Enable Developer mode in the top right corner.
4
LOAD THE EXTENSION
Tap + (from .zip) and select the downloaded file.
The extension will install automatically.
5
ACTIVATE YOUR KEY
Open TikTok, tap the extension icon.
